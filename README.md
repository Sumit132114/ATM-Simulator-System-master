📌 Bank Management System (ATM Simulator System)

A Java-based ATM Simulator System that replicates real-world banking operations. This project demonstrates core concepts of Java, Object-Oriented Programming (OOP), and database integration by simulating how an ATM machine works.

🚀 Project Description

The Bank Management System (ATM Simulator) is a console/GUI-based application developed in Java that allows users to perform basic banking operations just like a real ATM.

It provides a secure and user-friendly interface where users can:

Create a new bank account
Deposit money
Withdraw cash
Check account balance
View mini statements
Change ATM PIN

This project is ideal for learning Java, JDBC, and database handling, and showcases how real banking systems manage transactions.

🛠️ Tech Stack
Language: Java
Concepts Used: OOP (Encapsulation, Inheritance, Polymorphism)
Database: MySQL / Oracle (via JDBC)
IDE: IntelliJ IDEA / Eclipse / NetBeans
UI: Swing / AWT (if GUI-based)
✨ Features
🏦 Account Creation – Open a new bank account with user details
💰 Deposit Money – Add funds securely
💸 Withdraw Money – Withdraw with balance validation
📊 Mini Statement – View transaction history
🔐 PIN Change – Update ATM PIN securely
💳 Balance Inquiry – Check current balance instantly
🔒 Security Features
PIN-based authentication
Input validation
Basic protection against invalid transactions
